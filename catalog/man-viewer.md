# man-viewer — Documentation Viewer

Desktop viewer for man/ documentation generated by rulestools documenter.
Slint UI with search, tree navigation, and coverage display.

> Repo: `D:/REPO/Rules-dev/ManViewer` (local)

---

## What It Provides

| Feature | Description |
|---------|-------------|
| Tree view | Navigate crates → files → items |
| Search | Full-text search across all documented items |
| Coverage | Visual doc coverage percentage per file |
| MANIFEST | Reads man/MANIFEST.json generated by `rulestools gen` |

---

## Usage

```bash
# Generate docs first
rulestools gen .

# Open viewer
man-viewer man/
```
