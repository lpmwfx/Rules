# AUTHORS.md File

> Contributors to the project — human and AI

---

## Quick Reference

- **Location:** `proj/AUTHORS.md`
- **Format:** Markdown — `## Human` and `## AI Assistants` sections
- **Required:** Always
- **Owner:** User manages — update when contributors change

Lists all contributors with their roles. Credit both human and AI contributors.
