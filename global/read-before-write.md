# Read Before Write

> Never modify files without reading them first

---

VITAL: NEVER write to a file you haven't read in this session
VITAL: Read the ENTIRE file, not just a snippet
VITAL: Understand what exists before changing anything
REASON: AI cannot reliably remember file contents — always verify
